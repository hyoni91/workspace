배열 다루기

    자바스크립트는 배열과 리스트의 개념이 동일하다

    [특징]
    1. 자료형이 달라도 저장이 가능하다.
    2. 몇 개 저장할 지 정하지 않아도된다. (자바의 리스트와 동일)
    3. 위에서 아래로 실행

    ★for each 추천 ★
    element : 배열의 데이터 값 , index : 배열의 요소 
    매개변수 이름은 바꿀 수 있음 (e , i) 도 가능!

    arr.forEach(function(element, index){
        console.log(`element=${element} / index=${index}`);
    });
