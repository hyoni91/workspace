구조 분해 할당

    용어 정리 

    - 배열(리스트) : '[]'
    - 객체 : '{}'
    - 함수 : ()

    {객체 용어}
    const 객체명 = {
    Key : value,
    Key : value,
    ....
    };

    [구조 분해 할당]
    -배열과 객체에 사용하는 문법 / 리액트에서 자주 사용

    //배열 분해 할당
    -배열은 순번이 존재
     그렇기 때문에,  const [a1,b1,c1] = arr2; 의 경우 arr2의 차례대로 각각의 변수에 저장된다
    -배열 분해 할당의 경우 배열을 나타내는 [] 안에 변수를 선언한다.

    //객체 분해 할당
    -객체는 key : value 가 존재
    -구조 분해 할당을 받을 변수는 객체의 key값과 일치해야 한다.
    -구조 분해 할당의 경우 객체를 나타내는 {} 안에 변수를 선언한다.
    const {name,age,tle} = member;  key값 동일하게 해야함
    console.log(`name = ${name}`);

    -부득이 하게 key값 이름을 바꿀 경우?  잘 안써요~~~~ 그냥 이런게 있구나 하면 된당
    //객체의 구조 분해 할당 문법에서 변수명을 변경하는 문법
    const {stuNum : num , korScore : kor } = student;  //key값의 value를 : 이 이름으로 저장
    console.log(`num = ${num} , kor = ${kor}`);
